# Hey, I'm Iman 👋🏾

I'm a **Data Analyst** with a chemistry background and a deep curiosity
about what data can reveal when you ask it the right questions.
I work across the full pipeline — from messy raw data to polished
dashboards that stakeholders actually use to make decisions.

My tools of choice are **Python, SQL, Power BI, and Excel** —
but the real work is in the thinking before the tools.

## 🧠 What I Actually Do

I don't just build reports. I build answers.

- **Clean, integrate and model data** — ETL and EDA so that
  teams can trust their numbers before acting on them
- **Build predictive models** — classification and forecasting
  in Python, with explainability built in from the start
- **Design dashboards** that convert analysis into decisions —
  not just visuals, but narratives that non-technical stakeholders
  can pick up and run with
